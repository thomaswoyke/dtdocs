correção da lente

Corrige automaticamente (e simula) a distorção da lente, aberrações cromáticas transversais (ACT) e vinhetas.

Este módulo identifica a combinação de câmera/lente a partir dos dados Exif da imagem e usa a biblioteca externa lensfun para fornecer parâmetros de correção.

Se a biblioteca lensfun do sistema não tiver nenhum perfil de correção para a combinação automática de câmera/lente identificadas, os controles para os três parâmetros fotométricos (abaixo) são substituídos por uma mensagem de aviso. Você então pode tentar encontrar o perfil correto pesquisando no menu.

Se a sua lente estiver presente na lista, mas não foi identificada corretamente, pode ser necessário algum ajuste dentro do programa exiv2 (consulte este post para detalhes). Observe que você pode precisar re-importar as imagens uma vez que tais ajustes foram feitos para que o nome da lente recuperada erroneamente seja relido como parte do processo de importação.

Se você não conseguir encontrar sua lente, verifique se ela está na lista de lentes atualmente suportadas e tente executar a ferramenta lensfun-update-data. Se ainda não houver perfil correspondente para sua lente, um serviço de calibração de lente é oferecido por Torsten Bonger, um dos usuários do darktable. Como alternativa, você pode visitar o projeto lensfun para aprender a gerar seu próprio conjunto de parâmetros de correção. Não se esqueça de compartilhar seu perfil com a equipe da lensfun!

🔗controles do módulo

câmera
A marca e modelo da câmera, conforme determinado pelos dados Exif da imagem. Você pode substituir isso manualmente e selecionar sua câmera através de um menu hierárquico. Serão mostradas apenas lentes com perfis de correção que correspondam à câmera selecionada.
lente
A marca e modelo da lente, conforme determinado pelos dados Exif da imagem. Você pode substituir isso manualmente e selecionar sua lente através de um menu hierárquico. Isto é principalmente necessário para lentes mecânicas puras, mas também pode ser necessário para lentes fora do mercado/de terceiros.
parâmetros fotométricos (comprimento focal, abertura, distância focal)
As correções da lente dependem de determinados parâmetros fotométricos que são lidos a partir dos dados Exif da imagem: distância focal (para distorção, ACT, vinhetas), abertura (para ACT, vinheta) e distância focal (para vinheta). Muitas câmeras não registram a distância focal em seus dados Exif, caso no qual você precisará inserir isso manualmente.

Você pode substituir manualmente todos os parâmetros selecionados automaticamente. Selecione um dos valores predefinidos do menu suspenso ou, com o menu suspenso ainda aberto, apenas digite seus próprios valores.

correções
Escolha quais correções (distorção, ACT, vinhetas) devem ser aplicadas pelo darktable. Altere isso de seu padrão “todos” se sua câmera já tiver executado algumas correções internas (por exemplo, vinhetas), ou se você planeja realizar algumas correções com um programa separado.
geometria
Além de corrigir falhas da lente, este módulo pode alterar o tipo de projeção da sua imagem. Defina esta menu suspenso para o tipo de projeção desejado (por exemplo, “retilínea”, “olho-de-peixe”, “panorâmica”, “equiretangular”, “ortográfica”, “estereográfica“, “ângulo equisólido”, “olho de peixe thoby”). Para corrigir a proporção de uma lente anamórfica, use o módulo rotação e perspectiva.
escala
Ajusta o fator de dimensionamento da sua imagem para evitar cantos pretos. Pressione o botão “escala automática” (à direita do controle deslizante) para o darktable encontrar automaticamente o melhor ajuste.
modo
O comportamento padrão deste módulo é corrigir falhas de lentes. Altere para “distorcer” para, em vez disso, simular as falhas/distorções de uma lente específica (efeito invertido).
sobrescrever ACT
Marque esta caixa para substituir os parâmetros de correção automática para o ACT. Isso exporá os parâmetros ACT vermelho e azul abaixo. Desmarque a caixa para reverter para as correções automáticas.
ACT vermelho; ACT azul
Substitui os parâmetros de correção para ACT. Você também pode usar os controles deslizantes para definir manualmente os parâmetros se o perfil da lente não incluir correção ACT. Olhe para as costuras coloridas em recursos com bordas de alto contraste e ajuste os parâmetros ACT para reduzi-las.
correções feitas
Ocasionalmente, para uma determinada combinação de câmera/lente, apenas algumas das possíveis correções são suportadas pelos perfis da biblioteca lensfun. Esta caixa de mensagem dirá quais correções realmente foram aplicadas à imagem.

Nota: As correções ACT não serão aplicadas a imagens que foram identificadas como monocromático (consulte processar imagens monocromáticas para mais informações).

Nota: O módulo de correção da lente preencherá dados ausentes nas bordas, repetindo seus pixels. Para correções mais pesadas, este preenchimento pode ser visível (especialmente em imagens com muito ruído). Corte a imagem, se necessário.

mostrar linhas guia
Selecione a caixa para mostrar guias sobrepostas sempre que o módulo for ativado. Clique no ícone à direita para controlar as propriedades das guias. Consulte guias & sobreposições para mais detalhes.
